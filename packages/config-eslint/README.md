# Eslint

За основу взяты code style от Airbnb, https://github.com/airbnb/javascript и https://github.com/airbnb/javascript/tree/master/react, за некоторыми исключениями -

* файлы, содержащие jsx-элементы, имеют расширение `.jsx`
* точка с запятой в конце строки не ставится
* отступы в 4 пробела (включая jsx-элементы и их атрибуты)
  ** длина строки не ограничена
* нельзя ставить несколько более одной пустой строки (и одна в конце файла)
* можно ставить обработчики событий на статичные элементы (span, div итд)

Более подробно можно посмотреть в файле `.eslintrc`

## Linting
В проекте используется линтинг Eslint и повешен обработчик на коммит, не дающий закоммитить файлы, которые не прошли проверку. Исключения прописаны в файле `.eslintignore`
